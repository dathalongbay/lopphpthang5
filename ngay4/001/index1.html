Thuộc tính css margin

		The CSS margin properties are used to create space around elements, outside of any defined borders.

With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

		- Thuộc tính css margin được sử dụng để tạo khoảng không gian xung quanh 1 thẻ html
		bên ngoài đường viền
		- Margin có 4 cạnh khác nhau 
		+ margin-top ( khoảng cách bên trên thẻ html )
		+ margin-bottom ( khoảng cách bên dưới thẻ html )
		+ margin-left ( khoảng cách bên trái thẻ html )
		+ margin-right ( khoảng cách bên phải thẻ html )
