thuộc tính transform css
        translate(X, Y) => dich chuyển
        rotate() => xoay
        scale() => thay đổi kích cỡ
        skew() => làm nghiêng
        matrix() kết hợp của các kiểu bên trên

        matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())

scaleX => thay đổi kích thước theo chiều ngang
skewY => làm nghiêng theo chiều dọc
 skewX => làm nghiên theo chiều ngang
   translateX => dịch chuyển theo chiều ngang
    translateY => dịch chuyển theo chiều dọc
