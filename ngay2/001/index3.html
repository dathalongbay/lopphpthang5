Thuộc tính css trong thẻ html
		- Khác với thuộc tính id được sử dụng để dịnh danh 1 thẻ html 
		là duy nhất
		- Thuộc tính class được sử dụng để gom nhóm các thẻ html
		vào 1 nhóm để tiện cho việc style trang trí

		yêu cầu :
		style cho các tính miền bắc có chữ màu xanh
		miền trung có màu vàng
		miền nam có màu đỏ

		tên class phải viết liên và không có dấu

		chọn thẻ html trong css dựa vào thuộc tính class
		class="mienbac" trong css .mienbac
		class="mientrung" trong css .mientrung

		yêu cầu số 2 
		thêm màu nền cho các tỉnh thành thuộc trung ương
		màu nền tím cho các tp tỉnh thuộc trung ương 

		=> thêm nhiều class cho 1 thẻ html
		className1 className2 className3 className4 ...

		yêu cầu 
		cho ha nội là thủ đô có font chữ to hơn các tp khác
		bằng việc sử dụng class
