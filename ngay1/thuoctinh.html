Thuộc tính của thẻ html
		- Đinh nghĩa thuộc tính của thẻ html bổ sung tính chất ý nghĩa cho 1 thẻ html
		- Thuộc tính cú phap tên thuộc tính ="giá trị của thuộc tính"
		- GIá trị của thuộc tính html nằm trong nháy kép "" hoặc nháy đơn ''
		- Mỗi thẻ khác nhau thì có thuộc tính khác nhau
		html lang="en" có nghĩa là file html có ngôn ngữ mặc định là tiếng anh
		html lang="vi" có nghĩa là file html có ngôn ngữ mặc định là tiếng việt

		ví dụ thẻ a có thuộc tính href chứa liên kết của thẻ a
		thẻ img có thuộc tính src chứa đường dẫn của hình ảnh
