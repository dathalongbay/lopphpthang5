Thuộc tính position trong CSS
        position là vị trí trong css
        - static ( giá trị mặc định ) có nghĩa là tĩnh
        mặc định tất cả các html đều có
        position: static;

        - relative ( có nghĩa là tương đối )
        - absolute ( có nghia là tuyệt đối )
        - fixed ( có nghĩa là cố định )
        - sticky ( trung gian của relative và fixed )

        dùng 4 thuộc tính để đặt tọa độ
        top : vị trí bên trên
        left : vị trí bên trái
        right : vị trí bên phải
        bottom : vị trí bên dưới
    
        kết luận :
        khi đặt thẻ html có
        position: relative;
        thì thẻ html đó sẽ lấy vị trí ban đầu của nó
        làm gốc tọa độ để dịch chuyển
        nếu đặt top: 50px;
        thì thẻ html đó dịch chuyển xuống dưới 50px so với vị trí ban đầu của thẻ
        nếu đặt top: -50px;
        thì thẻ html đó dịch chuyển lên trên 50px so với vị trí ban đầu
        nếu đặt left: 100px; thì dịch chuyển sang phải 100px
        nếu đặt left: -100px; thì dịch chuyển sang trái 100px
box relative là vị trí ban đầu của relative
